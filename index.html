Como estarão as linguagens voltadas a Web em 20 anos

Por: Vitor Oliveira Martins

O que é linguagem de programação?

A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte - que pode ser compilado e transformado em um programa de computador, ou usado como script interpretado - que informará instruções de processamento ao computador.

Minha Visão Sobre - 1

No meu ver, umas das principais coisas que poderia ter, seria uma mair flexibilidade em relação à leitura de diferentes códigos, como exemplo, o fato do GitHub entender o código do Visual Studio Code, mas, se nós pormos no contrário, o Visual Studio Code não compreende um código feito no Github. Tendo assim uma maior ampliação na criação de códigos grandes e de desenvolvedores que precisam trabalhar juntos, mas usam de diferentes editores de código-fonte.

Minha Visão Sobre - 2

Outros pontos seriam do uso de IA’s tanto na criação de códigos quanto na parte de corrigir ou apontar como e quais linhas estão afetando os elementos de forma mais rápida e prática, e um meio de organização mais simples, como de exemplo: cobrir links muito extensos ou também sem precisar criar muitos arquivos .html, .css e etc. modificando separadamente ou junto e indo como funciona o css, uma parte onde você pode editar sem que apareça no código base

Conclusão

Por fim, mesmo não tendo um conhecimento tão vasto para dar ideias ou hipóteses de como sera, eu ainda sim adorarei ver a evolução das linguagens web aprendendo o que já foi deixado atrás como a história de como começou, quanto no presente utilizando na prática oque sei e fazendo códigos melhores com oque virei a aprender.
